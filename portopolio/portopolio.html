Halo! Saya Hervan

Halo! Nama saya Hervan Syah, siswa SMK Taruna Bhakti yang sangat suka belajar tentang dunia teknologi, khususnya pemrograman web. Saya senang mempelajari HTML, CSS, dan JavaScript karena ketiganya bisa digunakan untuk membuat website yang menarik dan interaktif. Dengan HTML, saya bisa membuat struktur dasar halaman web. Melalui CSS, saya belajar mendesain tampilan agar terlihat indah dan responsif. Lalu dengan JavaScript, saya bisa menambahkan logika dan interaksi supaya website terasa hidup dan dinamis. Saya ingin terus mengasah kemampuan saya agar bisa menjadi web developer profesional yang kreatif, inovatif, dan mampu menciptakan karya bermanfaat di dunia al.
